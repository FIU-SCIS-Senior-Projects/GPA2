FIU GPA TRACKER AND FORECASTER - Release 2

The FIU GPA Forecaster and Tracker is a website that allows Florida International University students to register, import their study program courses, enter grades in their currently enrolled courses and then give the student advice for what they need to achieve the GPA that they want. The system stores the data persistently so that when a new semester begins students will still have access to the information already entered. Also, the system will allow the students to define how hard they think a class is and how relevant it is to them personally and use these parameters to produce a forecast report which will advise the student on the minimum grade they need for each course to achieve the target GPA goal. The main objective of this website is to provide FIU students with a tool that can help them stay on track in their academic objectives.

THE GPA DASHBOARD

The GPA Dashboard allows students to track their academic progress throughout their time at FIU and compare their calculated GPA to the GPA required for their target graduate program. Students will have the option to receive an indiviualized forecast upon assigning a value to the weight and relevance of classes they are going to take. The forecast generates the appropriate grades for each class that a student should strive for.

THE SEMESTER DASHBOARD

The Semester Dashboard provides a way for students to keep track of how they are doing in a certain course. For each course a user can create a set of "Assessment Types" which are basically a collection of grades that have the same weight. One possible example is a course where homework is worth 30% of the grade. The user can then create an Assessment Type called "Homework" with a weight of 30%. The user can insert what grades they got for assessments in that Assessment Type. The system will persistently store information about grades and automatically calculate total running average for the whole semester.
